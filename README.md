# SERGIO ALEXO — Logo Package

Official logo assets for the **SERGIO ALEXO** brand.  
This repository contains the first version of the visual identity, designed for use across music, digital platforms, and promotional materials.

---

## ✨ Overview
This project is the foundation of the SRGIO ALEXO brand identity.  
It includes a set of logos and variations built to be clean, modern, and versatile.

---

## 📦 Contents
- Primary logo
- Alternate logo variations
- Icon / symbol versions
- Light & dark theme variants

---

## 🗂 File Formats
- **SVG** — scalable vector (recommended for most uses)
- **PNG** — transparent background
- **JPG** — preview / quick use

---

## 🚀 Usage
You can use these assets for:
- Social media branding
- Music releases & cover art
- Event promotion
- Video / DJ visuals
